[使い方]
1.マテリアルをオブジェクトに適用する
2.パラメーターを調整する
  Caramel Color : カラメルの色
  Pudding Color : プリンの色
  Background Color : 背景色
  Upper Surface Radius : 上面の半径
  Upper Surface Width : 上面の幅
  Upper Height : 上の高さ
  Lower Surface Radius : 底面の半径
  Lower Surface Width : 底面の幅
  Lower Height : 下の高さ
  Resolution : 解像度
  Pudding Size : プリンの大きさ
  Angle : 回転角度
  Shift : 配置をずらすかどうか
